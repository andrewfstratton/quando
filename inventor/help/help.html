Display

Display blocks provide a series of window action under specified circumstances defined by users. For example, showing the content and choices, or prompt out input box.

When display


When-display block means “When there shows something, then do something.”
In default, this block displays the input content as a title on the window to indicate the current state (similar to the condition of “if” – the title shows which condition you reach now). You can set trigger conditions for some actions by putting blocks inside this block.
If there exist label-to blocks, the input will be added as an option to label-to block automatically.

Label to


Label-to block is used to switch conditions. In label-to block, inputs of when-display blocks are the switching options. The selected option will display as a link related to the When-display block that has the same input content.
Should be noticed, there will be no option in the Label-to block if there is no When-display block in the script.

Media

Style

Style blocks can modify the style of the components that display on the code execution window, including background, title, text and label.

Text style


Text-style can change the text colour or its background colour. Text can from title, text or label.

Time

Time blocks allow you to use time as a condition for performing actions.

Every


This block indicates “do something in every time scale”.  This block only can be dragged into the When-display block.
